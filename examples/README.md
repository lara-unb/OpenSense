## Exemplos

Esta pasta contém scripts exemplos para utilização das funções do projeto LARA Opensense.

Os exemplos são:

1. ```json_to_sto.py```: Exemplo de conversão de um arquivo de captura de dados de IMU ```.json``` em arquivo ```.sto``` para utilização do OpenSense.
2. ```joint_angles.py```: Exemplo de obtenção de ângulos de articulações e impressão dos dados em um gráfico com e sem filtro de dados (filtro butterworth de segunda ordem).
