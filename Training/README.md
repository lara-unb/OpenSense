# Tutoriais

Tutoriais destinados ao treinamento de novos membros do projeto:

1. [__Tutorial Básico__](TutorialBasico/): Deve ser o primeiro tutorial do treinamento e é destinado a explicar o procedimento para processamento de dados de IMUs dentro a interface do OpenSim (Utilizando as funcionalidades do OpenSense)
2. [__Cinemática Inversa__](CinematicaInversa/): Atividade prática para entendimento do processamento de dados de orientação para obter a cinemática inversa do movimento capturado.
